Semantic Tag

Semantic Tags provide meaning to the structure of web content. They tell both the browser and developer what kind of content is being presented.
They enhance SEO, improve accessibility, and make our code easier to read and maintain.

Some key semantic tags:

  1. <header>: represents the topsection of web page, often containing logos, headings, and navigation.
  2. <nav>: Signifies a navigation menu on a web page.
  3. <article>: indicates a self-contained piece of content, like blog post or news article.
  4. <section>: thematic grouping of content on a web page.
  5. <aside>: used for sidebars or content related to main content.
  6. <footer>: represents footer of the web page, usually contains copyright and contact details.
  7. <figure> and <figcaption>: embedding images, diagrams, or charts, along with a coupon.
  8. <main>: represents main content area of a web page.
  9. <time>: represents time-related information like dates and time.
